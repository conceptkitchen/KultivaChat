Logout Functionality Test

Important: The logout buttons in the application work correctly with POST requests.

The "Method Not Allowed" error occurs when you manually type /api/logout in the browser address bar, which sends a GET request instead of using the logout buttons.

How to properly logout:

  1. Use the logout button in the header (top right corner)
  2. OR use the logout button in the sidebar (bottom left)

Do NOT:

  • Type "/api/logout" in the browser address bar
  • Navigate directly to the logout URL

Both logout buttons use proper POST mutations and will redirect you to the landing page successfully.
